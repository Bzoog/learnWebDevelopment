العناصر الأساسية في HTML ووظيفة كل منها

اليوم الأول

1. html

الوصف: الجذر لكل محتوى صفحة الويب.

1. html

Description: The root of all content in a web page.

2. head

الوصف: معلومات غير مرئية عن الصفحة (العنوان، الروابط).

2. head

Description: Contains non-visible information about the page (title, links).

3. body

الوصف: المحتوى المرئي للصفحة (نصوص، صور).

3. body

Description: The visible content of the page (texts, images).

4. meta

الوصف: معلومات وصفية عن الصفحة (ترميز، وصف).

4. meta

Description: Descriptive information about the page (encoding, description).

5. h1 ... h6

الوصف: عناوين المحتوى:

  • h1: العنوان الرئيسي (واحد فقط).
  • h2: العناوين الثانوية.
  • h3: العناوين الفرعية.
  • h4: عناوين أكثر تفصيلًا.
  • h5: عناوين تحت h4.
  • h6: أقل العناوين أهمية.

5. h1 ... h6

Description: Content headings:

  • h1: Main title (only one).
  • h2: Secondary headings.
  • h3: Sub-headings.
  • h4: More detailed headings.
  • h5: Headings under h4.
  • h6: The least important heading.

6. p

الوصف: فقرة نصية.

6. p

Description: A paragraph of text.

اليوم الثاني

1. b

الوصف: يُستخدم لتنسيق النص بالخط العريض.

1. b

Description: Used to make text bold.

2. i

الوصف: يُستخدم لتنسيق النص بالخط المائل.

2. i

Description: Used to italicize text.

3. u

الوصف: يُستخدم لتسطير النص.

3. u

Description: Used to underline text.

4. del

الوصف: يُستخدم للإشارة إلى النص المحذوف.

4. del

Description: Used to indicate deleted text.

5. mark

الوصف: يُستخدم لتسليط الضوء على النص.

5. mark

Description: Used to highlight text.

6. a

الوصف: يُستخدم لإنشاء روابط تشعبية.

6. a

Description: Used to create hyperlinks.

7. img

الوصف: يُستخدم لإضافة الصور إلى الصفحة.

7. img

Description: Used to add images to a page.

8. span

الوصف: يُستخدم لتجميع النصوص أو العناصر دون إضافة أي تنسيق.

8. span

Description: Used to group inline elements without applying any formatting.

9. strong

الوصف: يُستخدم لتنسيق النص ليظهر بشكل أكثر قوة، وعادةً ما يُعرض بخط عريض للدلالة على أهمية النص.

9. strong

Description: Used to format text to appear stronger, and it is typically displayed in bold to indicate the importance of the text.

10. em

الوصف: يُستخدم لتنسيق النص ليظهر بشكل مائل، وعادةً ما يُعبر عن تأكيد أو تمييز في المعنى.

10. em

Description: Used to format text to appear italicized, often conveying emphasis or distinction in meaning.

11. small

الوصف: يُستخدم لتقليل حجم النص، وعادةً ما يُستخدم للإشارة إلى نص ثانوي أو ملاحظات.

11. small

Description: Used to decrease the font size of the text, typically used to indicate secondary text or notes.

الخصائص الأساسية في HTML ووظيفة كل منها

1. src

الوصف: تُستخدم لتحديد مصدر الصورة في عنصر <img>.

Basic HTML Attributes and Their Functions

1. src

Description: Used to specify the source of an image in the <img> element.

2. alt

الوصف: تُستخدم لتوفير نص بديل للصورة في حالة عدم تحميلها.

2. alt

Description: Used to provide alternative text for an image if it fails to load.

3. href

الوصف: تُستخدم لتحديد عنوان الرابط في عنصر <a>.

3. href

Description: Used to specify the URL of the link in the <a> element.

4. target

الوصف: تُستخدم لتحديد كيفية فتح الرابط (مثل فتحه في نافذة جديدة أو نفس النافذة) في عنصر <a>.

4. target

Description: Used to specify how to open the link (e.g., in a new window or the same window) in the <a> element.

اليوم الثالث

1. div

الوصف: يُستخدم كحاوية عامة لتجميع العناصر وتطبيق الأنماط أو تنظيم المحتوى. لا يضيف أي تنسيق افتراضي.

1. div

Description: Used as a general container to group elements and apply styles or organize content. It does not add any default styling.

2. main

الوصف: يُستخدم لتحديد المحتوى الرئيسي للصفحة، يساعد في تحسين تنظيم المحتوى وتجربة المستخدم.

2. main

Description: Used to define the main content of the page, helping to improve content organization and user experience.

3. section

الوصف: يُستخدم لتقسيم المحتوى إلى أقسام منطقية، حيث يمكن أن يحتوي كل قسم على عنوان ومحتوى ذي صلة.

3. section

Description: Used to divide content into logical sections, where each section can contain a heading and related content.

4. header

الوصف: يُستخدم لتحديد رأس الصفحة أو قسم معين، وعادةً ما يحتوي على عناصر مثل العناوين والشعارات.

4. header

Description: Used to define the header of a page or a specific section, typically containing elements like headings and logos.

5. footer

الوصف: يُستخدم لتحديد تذييل الصفحة أو قسم معين، وعادةً ما يحتوي على معلومات حقوق النشر، وروابط، ومعلومات الاتصال.

5. footer

Description: Used to define the footer of a page or a specific section, typically containing copyright information, links, and contact details.

6. nav

الوصف: يُستخدم لتحديد مجموعة من الروابط المستخدمة للتنقل في الصفحة، مما يسهل على المستخدمين الوصول إلى أقسام مختلفة من الموقع.

6. nav

Description: Used to define a set of links used for navigation within the page, making it easier for users to access different sections of the site.

الخصائص الأساسية في HTML ووظيفة كل منها

1. id

الوصف: تُستخدم لتحديد معرف فريد لعنصر في الصفحة، مما يسمح بالوصول إليه بسهولة باستخدام CSS أو JavaScript.

Basic HTML Attributes and Their Functions

1. id

Description: Used to specify a unique identifier for an element on the page, allowing easy access via CSS or JavaScript.

2. class

الوصف: تُستخدم لتحديد فئة أو مجموعة من العناصر، مما يسهل تطبيق أنماط CSS أو التعامل مع العناصر باستخدام JavaScript.

2. class

Description: Used to specify a class or group of elements, making it easier to apply CSS styles or manipulate elements with JavaScript.

3. hidden

الوصف: تُستخدم لإخفاء عنصر في الصفحة، حيث يتم تجاهله في عرض الصفحة دون حذفه من الشيفرة.

3. hidden

Description: Used to hide an element on the page, where it is ignored in the rendering without being removed from the markup.

4. dir

الوصف: تُستخدم لتحديد اتجاه النص، سواء كان من اليسار إلى اليمين (ltr) أو من اليمين إلى اليسار (rtl).

4. dir

Description: Used to specify the direction of text, either left-to-right (ltr) or right-to-left (rtl).

5. lang

الوصف: تُستخدم لتحديد لغة المحتوى في العنصر، مما يساعد محركات البحث وأدوات الوصول في فهم اللغة المستخدمة.

5. lang

Description: Used to specify the language of the content within the element, helping search engines and accessibility tools understand the language used.

6. title

الوصف: تُستخدم لتوفير نص تفسيري يظهر عند تمرير الماوس فوق العنصر، مما يساعد المستخدمين في فهم محتوى العنصر بشكل أفضل.

6. title

Description: Used to provide explanatory text that appears when hovering over the element, helping users better understand its content.

7. style

الوصف: تُستخدم لتطبيق أنماط CSS مباشرة على العنصر، مما يسمح بتخصيص مظهر العنصر بشكل فوري.

7. style

Description: Used to apply CSS styles directly to the element, allowing for immediate customization of its appearance.

اليوم الرابع

1. ul

الوصف: يُستخدم لإنشاء قائمة غير مرتبة، حيث تكون العناصر فيها مسبوقة برموز مثل النقاط.

1. ul

Description: Used to create an unordered list, where items are preceded by symbols like bullets.

2. ol

الوصف: يُستخدم لإنشاء قائمة مرتبة، حيث يتم ترقيم العناصر تلقائيًا، مما يشير إلى ترتيبها.

2. ol

Description: Used to create an ordered list, where items are automatically numbered, indicating their sequence.

3. li

الوصف: يُستخدم لتعريف عنصر في القائمة، سواء كانت مرتبة أو غير مرتبة.

3. li

Description: Used to define an item in a list, whether it is ordered or unordered.

4. hr

الوصف: يُستخدم لإنشاء خط أفقي يفصل بين محتويات الصفحة، مما يساعد في تنظيم المعلومات بصريًا.

4. hr

Description: Used to create a horizontal line that separates content on the page, helping to organize information visually.

5. br

الوصف: يُستخدم لإدراج فاصل سطر، مما يُتيح الانتقال إلى السطر التالي دون إضافة مسافة إضافية.

5. br

Description: Used to insert a line break, allowing the text to continue on the next line without adding extra space.

الخصائص الأساسية في HTML ووظيفة كل منها

1. value

الوصف: تُستخدم لتحديد القيمة الحالية لعنصر إدخال (مثل مربع النص أو زر الاختيار). تُستخدم في النماذج لتحديد البيانات التي سيتم إرسالها عند تقديم النموذج.

Key Attributes in HTML and Their Functions

1. value

Description: Used to specify the current value of an input element (such as a text box or checkbox). It is used in forms to determine the data that will be sent when the form is submitted.

2. type

الوصف: تُستخدم لتحديد نوع عنصر الإدخال، مثل "text" للنص، و"checkbox" لصناديق الاختيار، و"submit" لأزرار الإرسال. يساعد هذا في تحسين تجربة المستخدم من خلال تخصيص سلوك العنصر.

2. type

Description: Used to specify the type of input element, such as "text" for text fields, "checkbox" for checkboxes, and "submit" for submit buttons. This helps enhance user experience by customizing the element's behavior

خصائص في لغة CSS (إضافية)

الخصائص الأساسية في CSS ووظيفة كل منها

1. color

الوصف: تُستخدم لتحديد لون النص داخل العناصر. يمكن أن يكون اللون محددًا بأسماء الألوان، أو قيم HEX، أو RGB.

2. background-color

الوصف: تُستخدم لتحديد لون خلفية العنصر. مثل خاصية color، يمكن استخدام أسماء الألوان أو قيم HEX أو RGB.

3. padding

الوصف: تُستخدم لتحديد المسافة بين محتوى العنصر وحدوده. يضيف الحشو داخل العنصر نفسه، مما يزيد من المساحة حول المحتوى.

4. margin

الوصف: تُستخدم لتحديد المسافة بين العنصر والعناصر المحيطة به. يضيف الهوامش خارج العنصر، مما يخلق فراغًا بينه وبين العناصر الأخرى.

Basic CSS Properties and Their Functions

1. color

Description: Used to specify the color of the text within elements. The color can be defined using color names, HEX values, or RGB.

2. background-color

Description: Used to specify the background color of an element. Similar to the color property, color names, HEX values, or RGB can be used.

3. padding

Description: Used to define the space between the content of the element and its borders. Padding adds space inside the element itself, increasing the area around the content.

4. margin

Description: Used to define the space between the element and surrounding elements. Margin adds space outside the element, creating a gap between it and other elements.
