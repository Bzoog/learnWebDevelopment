اليوم الثالث: حقول إدخال القيم من المستخدم

في هذا الدرس، سنتعرف على كيفية إنشاء نماذج تفاعلية (Forms) تُستخدم لجمع بيانات المستخدم، مع شرح تفصيلي للوسوم والخصائص المرتبطة بها، مدعومة بالأمثلة العملية.

الوسوم الأساسية لإنشاء النماذج

لتصميم النماذج، نستخدم الوسوم التالية: ينبغي استخدام الوسوم بشكل صحيح لضمان التوافقية وإمكانية الوصول.

1. الوسم Form

يُستخدم لتعريف النموذج ككل. يمكن أن يحتوي على وسوم إدخال بيانات (input) ووسوم أخرى.


<form action="/submit-data">
    
</form>

2. الوسم Label

يُستخدم لربط وصف نصي بحقل إدخال لزيادة قابلية الاستخدام.


<label for="username">اسم المستخدم:</label>
<input type="text" id="username" name="username">

3. الوسم Input

يُستخدم لإنشاء حقول إدخال مختلفة، مثل النصوص والأرقام وكلمات المرور. نوع الإدخال يتم تحديده باستخدام الخاصية type. 


<input type="text" placeholder="أدخل اسمك">

الخصائص المرتبطة بالوسوم

كل وسم يمكن تخصيصه باستخدام خصائص مختلفة. الخصائص تجعل النماذج أكثر ديناميكية وتفاعلية.

خصائص الوسم Input

الخاصية الوصف
type تحدد نوع الإدخال (مثل نص أو رقم).
placeholder نص إرشادي يظهر داخل الحقل عندما يكون فارغًا.
value تحدد قيمة افتراضية للحقل.
min و max يُستخدمان لتحديد القيم الدنيا والعليا للإدخال.

خصائص الوسم Form

الخاصية الوصف
action تحدد رابط الصفحة التي ستُرسل إليها البيانات بعد الإرسال.

خصائص الوسم Label

الخاصية الوصف
for تربط الوسم label بحقل إدخال باستخدام معرف (id).

القيم اللازمة للخاصية Type

الخاصية type تدعم عدة قيم، مثل:

القيمة الوصف
text حقل نصي لإدخال النصوص.
number حقل لإدخال الأرقام فقط.
submit زر لإرسال البيانات.
date حقل لاختيار التواريخ.
password حقل لإدخال كلمات المرور (تظهر كنقاط).

مثال عملي


<form action="/submit-data">
    <label for="email">البريد الإلكتروني:</label>
    <input type="text" id="email" placeholder="example@mail.com">

    <label for="age">العمر:</label>
    <input type="number" id="age" min="18" max="100">

    <label for="dob">تاريخ الميلاد:</label>
    <input type="date" id="dob">

    <label for="password">كلمة المرور:</label>
    <input type="password" id="password" placeholder="********">

    <input type="submit" value="إرسال">
</form>
