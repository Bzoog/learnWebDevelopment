كتابة التنسيقات في CSS

أماكن كتابة تنسيقات CSS

هناك ثلاث طرق أساسية لكتابة تنسيقات CSS

كتابة التنسيقات inline

يتم كتابة التنسيقات مباشرة داخل وسم HTML باستخدام الخاصية style. تُستخدم هذه الطريقة لتطبيق التنسيق على عنصر معين.

<!DOCTYPE html>
   <html lang="en">
   <head>
         <title>مثال على CSS Inline</title>
   </head>
   <body>
   <p style="color: red; background-color: yellow;">هذا نص بتنسيق Inline</p>
   </body>
   </html>

كتابة التنسيقات In Head

يمكنك وضع التنسيقات داخل وسم <style> في القسم <head> من مستند HTML.


<!DOCTYPE html>
<html lang="en">
<head>
      <title>مثال على CSS في الرأس</title>
      <style>
         p {
            color: blue;
            background-color: lightgray;
         }
      </style>
</head>
<body>
      <p>هذا نص بتنسيق داخل الرأس</p>
</body>
</html>

كتابة التنسيقات في ملف خارجي (CSS File)

أفضل الممارسات هي كتابة التنسيقات في ملف خارجي، حيث يتم استدعاء الملف باستخدام وسم <link> داخل القسم <head>.

ملف HTML


<!DOCTYPE html>
<html lang="en">
<head>
    <title>مثال على CSS في ملف خارجي</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <p>هذا نص بتنسيق من ملف خارجي</p>
</body>
</html>

ملف CSS (styles.css)


p {
   color: green;
   background-color: lightyellow;
}

محددات الوصول في CSS

تحدد هذه المحددات العناصر التي يتم تطبيق التنسيقات عليها. تشمل الأنواع الأساسية

محدد العنصر (Tag Name)

يستهدف جميع العناصر ذات الاسم المحدد.


h1 {
   color: navy;
}

محدد الصنف (Class)

يستهدف العناصر التي تحتوي على قيمة محددة في الخاصية class.

<p class="highlight">هذا نص بتنسيق باستخدام صنف</p>

تنسيق css

.highlight {
   color: orange;
   background-color: black;
}

محدد المعرف (ID)

يستهدف عنصرًا فريدًا يحتوي على خاصية id.

<p id="unique">هذا نص بتنسيق باستخدام معرف

تنسيق css

#unique {
   color: purple;
   font-weight: bold;
}

العنصر الوصف
Inline تنسيق داخل العنصر باستخدام الخاصية style.
In Head تنسيق داخل وسم <style> في القسم <head>.
CSS File تنسيق في ملف خارجي يتم ربطه بـ <link>.
Tag Name استهداف العناصر باستخدام اسم الوسم.
Class استهداف العناصر باستخدام الخاصية class.
ID استهداف عنصر فريد باستخدام الخاصية id.
