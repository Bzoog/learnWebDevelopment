اليوم الثاني: تابع الخصائص الأساسية ووحدات القياس

في هذا الدرس، سنتعرف بشكل مفصل على الخصائص الأساسية للتحكم في أبعاد ومسافات العناصر، بالإضافة إلى وحدات القياس المختلفة المستخدمة في CSS لتحديد الأحجام والزوايا. سنوضح كيف تعمل كل خاصية مع أمثلة عملية وشروحات مبسطة.

الخصائص الأساسية

الخصائص الأساسية تساعد في تنسيق الأبعاد والمسافات حول العناصر، مما يسمح بتخصيص تصميم العناصر بطريقة أكثر مرونة. إليك الخصائص بالتفصيل: من المهم تجربة هذه الخصائص عمليًا لفهم كيفية تأثيرها على العنصر.

1. خاصية Margin

تُستخدم لتحديد المسافة الخارجية حول العنصر (المساحة بين العنصر والعناصر الأخرى). يمكن تحديد قيم مختلفة لجميع الاتجاهات.

  
  /* مثال على margin */
  div {
      margin: 10px; /* مسافة خارجية لجميع الجوانب */
      margin-top: 20px; /* مسافة من الأعلى */
      margin-right: 15px; /* مسافة من اليمين */
      margin-bottom: 10px; /* مسافة من الأسفل */
      margin-left: 5px; /* مسافة من اليسار */
  }

2. خاصية Padding

تُستخدم لتحديد المسافة الداخلية بين محتوى العنصر وحدوده.

  
  /* مثال على padding */
  div {
      padding: 10px; /* مسافة داخلية لجميع الجوانب */
      padding-top: 20px; /* مسافة من الأعلى */
      padding-right: 15px; /* مسافة من اليمين */
      padding-bottom: 10px; /* مسافة من الأسفل */
      padding-left: 5px; /* مسافة من اليسار */
  }

3. خاصية Border

تُستخدم لإنشاء حدود حول العنصر. يمكن تحديد السمك والنمط واللون.

  
  /* مثال على border */
  div {
      border: 2px solid black; /* حدود سوداء بلون صلب */
      border-width: 4px; /* سمك الحدود */
      border-style: dashed; /* نمط الحدود (متقطعة) */
      border-color: red; /* لون الحدود */
  }

4. خاصية Border-Radius

تُستخدم لتقويس زوايا العنصر.

  
  /* مثال على border-radius */
  div {
      border: 1px solid black;
      border-radius: 10px; /* تقويس جميع الزوايا */
      border-top-left-radius: 20px; /* تقويس الزاوية العليا اليسرى */
      border-bottom-right-radius: 15px; /* تقويس الزاوية السفلى اليمنى */
  }

5. خاصيتا Width و Height

تُستخدم لتحديد عرض وارتفاع العنصر.

  
  /* مثال على width و height */
  div {
      width: 300px; /* عرض العنصر */
      height: 200px; /* ارتفاع العنصر */
  }

وحدات القياس

CSS توفر مجموعة واسعة من وحدات القياس لتحديد الأحجام والزوايا: استخدم وحدات القياس النسبية مثل em وrem لتحسين مرونة التصميم.

أنواع وحدات القياس

الوحدة الوصف
px بيكسل - وحدة ثابتة لقياس الأحجام.
em نسبة إلى حجم الخط الخاص بالعنصر الأب.
rem نسبة إلى حجم الخط الخاص بعنصر الجذر (HTML).
deg الدرجات - تُستخدم مع التدوير والتحويلات.
vw 1% من عرض نافذة العرض (Viewport).
vh 1% من ارتفاع نافذة العرض (Viewport).

أمثلة على وحدات القياس

  
  /* مثال على وحدات القياس */
  div {
      font-size: 16px; /* بيكسل */
      margin: 1em; /* نسبة إلى الخط */
      padding: 1rem; /* نسبة إلى الجذر */
      transform: rotate(45deg); /* الدرجات */
      width: 50vw; /* نسبة إلى عرض النافذة */
      height: 50vh; /* نسبة إلى ارتفاع النافذة */
  }
